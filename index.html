HTML

  • The head element contains infor about the webpage.
  • The head element contains the metadata
  • Metadata attributes include description, keywords, anchor, author, viewport, http-equiv
  • Head element contains the Title tag (needed for SEO)
  • Semantic HTML elements - name of element describes the content and role eg head, body (not div)
  • Body tag contains header, main and footer tags
  • Use section tag to separate content in body

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Padding and margin can add to the overall size of the element. To stop this use box-sizing
  • Padding margin values can be assigned clockwise or top/bottom and right/left or using 3 options
  • box-shadow attribute can be used to give a shadow effect around an element. Assign values for X axis, Y axis, blur-radius, spread-radius, and colour

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds all changes in current working branch to the staging area
  • git commit - m "message" : captures snapshot of currently staged changes and commits
  • git pull origin main: pull the latest version of main into the local branch (origin means sourse is GitHub, main means main branch)
  • git push origin feature/add-git: push local changes to remote GitHub server

Javascript

  • Arrays - can store data of different types.
  • falsy - false, 0, -0, 0n, on, "", '', ``, null, undefined, NaN, document.all
  • truthy - everything else
